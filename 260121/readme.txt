##대문자로 시작하는 객체
System.out.println(), Integer.parseInt()함수에서 처음시작하는 객체가 대문자인 객체들은 함수가 static으로 선언된 객체이다.
이 객체는 별도의 객체를 생성하지 않고 바로 사용이 가능하다.
Integer.parseInt() : 문자열을 정수를 변환하는 함수

참고)객체를 만든 후 실행이 정상적이지 않을 경우 컴파일한 후 확인

##클래스(객체) 정의
class 객체명 { } 

##클래스 선언
객체명 변수명=new 객체명();
new 객체를 생성해야 실제 메모리에 업로드 됨. 이를 인스턴스라고 함.

##클래스의 형태
속성(변수)+기능(함수)

##public과 private
public : 모든 클래스에서 사용가능
private : 자신의 클래스에서만 사용 가능

##데이터를 전달하는 클래스에 정의할 함수
기본생성자함수, 필드생성자함수, get함수, set함수, toString함수

##배열의 필요성
객체명[] 변수명=new 객체명[100];

#객체를 만드는 방법
전화번호부를 구현하는 것이 최종적인 객체
하나의 전화번호를 저장하는 공간은 최소의 저장 단위
최소의 저장 단위를 배열로 묶으면 여러개의 전화번호를 저장할 수 있고 이를 통해 입력, 출력,수정,삭제기능이 내포되는 객체
이를 실제 실행하는 객체는 Main함수에서 구현
